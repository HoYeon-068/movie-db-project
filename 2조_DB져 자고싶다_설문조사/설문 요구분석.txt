메인 페이지:
1.설문조사는 번호로 분류한다
2.각 설문조사는 여러 개의 항목으로 구성된다
3.사용자는 메인 페이지에서 설문 조사 질문과 문항을 볼 수 있다
4.설문 목록 페이지에 들어가지 않아도 메인 페이지에서 바로 문항 선택으로 투표가 가능하고 결과도 볼 수 있다
단 최신 설문 하나만 가능하다




설문 목록 페이지:
1.사용자는 설문 목록을 조회할 수 있다
2.설문목록은 번호로 식별한다
3.설문목록에 들어갔을때 대한 
4.번호, 제목, 작성자, 시작일과 종료일,항목수,참여자수,상태를 알 수 있어야 한다
5.종료일을 기준으로 진행중인지 종료인지 상태 확인 가능하다
6.질문을 클릭하면 설문 상세 보기 페이지로 이동 가능하다


설문 작성 페이지:
1.사용자는 질문 입력, 설문 가능한 시작일과 종료일 선택, 항목수 선택, 
 항목 입력 후 확인을 통해 설문을 작성할 수 있다
2.관리자는 설문을 수정,삭제 할 수 있다



설문 상세 보기 페이지:
1.설문 작성 페이지에서 작성된 내용대로 
  질문, 작성자, 시작일, 종료일, 상태, 항목수, 항목 그리고 추가로 작성일까지 화면에 보여지고
  항목을 고른 후 투표하면 총 참여자 수와 설문 결과가 보여진다
